WolfScript

The base language is JavaScript, which not only powers the web but also is powering more and more servers worldwide, such as at Walmart, Paypal, Netflix and the Mail Online.

WolfNode

On top of JavaScript we've added a special implementation of Node.js which integrates the work of the "Project Odd" team at Red Hat to bring this to the Java-based environment of Minecraft.

WolfAPI

On top of JavaScript and WolfNode we provide a full MineCraft Server API.  We support the two most popular families of Minecraft servers:
WolfBuk for Bukkit/CraftBukkit/Spigot and WolfCanary for CanaryMod servers.

Learn to program, share your creations

Programming course included

Learning to program with WolfScript is easy and fun.   We'll start with the basics, like allowing each player to change their chat color.  

We'll build up to incorporating sensors, music players and light switches that work in the real world.  

The WolfNode ecosystem has over 130,000 packages at your disposal.

Create stand-alone plugins

The plugins created in WolfScript work on any standards-based server like Spigot or CanaryMod.

While we'll show you how to run your own servers and invite only your friends, or host them on Mining Wolf;  the mods you create are yours to keep. 

Open source your creations on GitHub, or package them up as a jar file and distribute to any server adminstrator -- they don't even need to know they're not written in Java.
