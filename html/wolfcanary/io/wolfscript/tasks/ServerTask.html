new ServerTask(owner, delay, continuous)

Constructs either a one-time use or continuous ServerTask

Argument Type Description
owner TaskOwner the TaskOwner of the task, typically a Plugin
delay long the delay before executing. Set to 0 or less to run within the next Server tick
If delay is 0 or less, the task will run with each server tick
continuous boolean true for continuous; false for one-time use
