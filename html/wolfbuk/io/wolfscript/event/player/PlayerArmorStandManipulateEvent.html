Get: Returns the item held by the player. If this Item is null and the armor stand Item is also null, there will be no transaction between the player and the armor stand. If the Player's item is null, but the armor stand item is not then the player will obtain the armor stand item. In the case that the Player's item is not null, but the armor stand item is null, the players item will be placed on the armor stand. If both items are not null, the items will be swapped. In the case that the event is cancelled the original items will remain the same.
