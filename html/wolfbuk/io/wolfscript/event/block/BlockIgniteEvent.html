BlockIgniteEvent class

io.wolfscript.event.block.BlockIgniteEvent Extends BlockEvent Implements Cancellable

this.on('block.BlockIgniteEvent', function (e) { });

Class Overview

Called when a block is ignited. If you want to catch when a Player places fire, you need to use BlockPlaceEvent.

If a Block Ignite event is cancelled, the block will not be ignited.
