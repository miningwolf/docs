HelpMap interface

io.wolfscript.help.HelpMap

Interface Overview

The HelpMap tracks all help topics registered in a WolfScript server. When the server starts up or is reloaded, help is processed and topics are added in the following order:

  1. General topics are loaded from the help.yml
  2. Plugins load and optionally call addTopic()
  3. Registered plugin commands are processed by HelpTopicFactory objects to create topics
  4. Topic contents are amended as directed in help.yml
